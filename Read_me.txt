# reactjs_calculator
Online Basic Calculator Using React.js
Open the command prompt:
1-  "node --v" to see your node.js version to be sure that node.js is installed
2-  "cd C:\Users\Idriss.Mortabit\Desktop\my-app" (in my case the folder exist in desktop)
3-  add the "node_modules" folder in "my-app" folder
4-  "npm start" or "npm start // eslint-desable-next-line "
5-  open your favorite browser and tap "http://localhost:3000/"
